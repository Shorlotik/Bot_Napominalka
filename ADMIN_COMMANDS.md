# Команды для администратора

Админ ID: `791152212`

## Базовые команды

- `/start` - Начать работу с ботом (доступна админу)
- `/admin` - Показать меню админ-команд

## Управление сообщениями

- `/add_message <текст>` - Добавить новое сообщение в список
  Пример: `/add_message дорогая не забывай про осанку`

- `/list_messages` - Показать список всех сообщений с их ID

- `/delete_message <id>` - Удалить сообщение по ID
  Пример: `/delete_message 5`

## Управление напоминалкой

- `/start_reminder` - Запустить напоминалку глобально для всех пользователей

- `/stop_reminder` - Остановить напоминалку глобально для всех пользователей

- `/set_schedule <start_hour> <end_hour>` - Изменить расписание работы
  Пример: `/set_schedule 9 19` (с 9:00 до 19:00)

## Статистика

- `/stats` - Показать статистику работы бота:
  - Количество сообщений в базе
  - Количество отправленных сообщений
  - Количество активных пользователей
  - Текущее расписание

## Справка

- `/help` - Показать меню админ-команд

## Примеры использования

```bash
# Добавить новое сообщение
/add_message любимая спину держи ровно я тебя обожаю

# Посмотреть все сообщения
/list_messages

# Удалить сообщение с ID 3
/delete_message 3

# Изменить расписание на 9:00-17:00
/set_schedule 9 17

# Посмотреть статистику
/stats

# Остановить напоминалку глобально
/stop_reminder
```
